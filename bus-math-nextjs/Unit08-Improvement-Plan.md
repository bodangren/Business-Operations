# Unit 08 Improvement Plan

## Status

This is a temporary planning note for scope being moved from Unit 07 into Unit 08.

Unit 07 is being refocused to inventory only.

Unit 08 should absorb the depreciation content that no longer belongs in Unit 07. This file is not a full redesign yet. It is a handoff list so the moved scope does not get lost.

## Scope moving from Unit 07 into Unit 08

- Capitalization versus expense treatment for long-term assets
- Asset cost allocation across useful life
- Depreciation vocabulary:
  - cost
  - salvage value
  - useful life
  - book value
  - accumulated depreciation
- Straight-line depreciation logic
- Double-declining balance depreciation logic
- Comparison of when a business might prefer straight-line versus double-declining balance
- Hand-built depreciation schedules before Excel automation
- Excel-based depreciation schedules and formula implementation
- Asset-register structure and fixed-asset tracking logic
- Workbook checks for depreciation outputs and documentation
- Recommendation prompts about which depreciation method best fits business goals

## Unit 07 content to remove or relocate

- Lesson 1 content that frames depreciation as a co-equal unit focus
- Lesson 2 depreciation-first lesson structure
- Any `SLN` and `DDB` explanation currently living in Unit 07 phase pages
- Any Unit 07 workbook assets whose main purpose is depreciation practice
- Any assessment questions in Unit 07 focused on depreciation methods rather than inventory methods
- Any closing copy in Unit 07 that previews depreciation-heavy technical work

## Likely Unit 08 insertion points

- Unit 08 Lesson 1 or Lesson 2:
  - introduce why startup models need defensible fixed-asset assumptions
- Early-to-middle Unit 08:
  - add depreciation schedules to startup operating model build
- Mid Unit 08:
  - connect depreciation to three-statement modeling and tax impact
- Later Unit 08:
  - include fixed-asset assumptions in scenario analysis and investor defense

## Minimum integration requirements for Unit 08

- Students should understand that depreciation is a modeling assumption with statement impact, not just a bookkeeping rule.
- Depreciation should feed income statement, balance sheet, and cash flow implications where relevant.
- Workbook structure should make fixed-asset assumptions visible and auditable.
- Method choice should be tied to startup context, not taught as abstract accounting trivia.

## Future redesign questions

- Should Unit 08 teach only straight-line and double-declining balance, or also mention tax/book differences?
- Should Unit 08 include a separate fixed-assets support sheet in the startup model workbook?
- Where should depreciation be assessed:
  - as a standalone lesson
  - or as part of integrated financial model accuracy
- Which current Unit 08 lessons can absorb this scope without overloading the unit

## Immediate next step

When Unit 07 rewrite work begins, remove depreciation-specific learning goals, activities, assessments, and assets from Unit 07 and use this file as the transfer checklist for the eventual Unit 08 update.
