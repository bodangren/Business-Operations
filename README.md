## 📚 Curriculum Overview

### Unit 1: Smart Ledger Launch
*How can we design a self-auditing ledger that would convince a potential angel investor we keep "clean books" from day 1?*
- Excel tables, SUMIF functions, conditional formatting
- **Deliverable**: 4-minute investor pitch with live demo

### Unit 2: Month-End Wizard  
*What automation can cut our month-end closing time from two days to two hours without sacrificing GAAP accuracy?*
- Macros, VBA, named ranges, adjusting entries
- **Deliverable**: Interactive Excel automation tool

### Unit 3: Three-Statement Storyboard
*How do today's journal entries flow into a narrative of profit, solvency, and cash health that investors can trust?*
- Cross-sheet linking, INDEX/MATCH, financial statement preparation
- **Deliverable**: Investor one-pager with KPI dashboard

### Unit 4: Data-Driven Café
*Given two years of POS data, what inventory and staffing plan will maximize weekend profits without raising waste above 3%?*
- Descriptive statistics, regression analysis, Excel Analysis ToolPak
- **Deliverable**: Management recommendation with forecast model

### Unit 5: PayDay Simulator
*How can a small business owner predict payroll cash-outs and still make rent on time?*
- XLOOKUP, payroll calculations, tax tables, reconciliation
- **Deliverable**: Tutorial screencast for young entrepreneurs

### Unit 6: PriceLab Challenge
*What pricing strategy hits our profit target while staying competitive in the local market?*
- Cost-Volume-Profit analysis, Goal Seek, sensitivity analysis
- **Deliverable**: Town-hall pricing debate presentation

### Unit 7: Inventory Accounting
*How do we build, test, and defend a trustworthy ending inventory number?*
- FIFO/LIFO inventory, weighted average, specific identification, turnover ratios
- **Deliverable**: Advisory brief to mock Board of Directors

### Unit 8: Fixed Assets and Depreciation
*How do we track long-term assets professionally and choose the right depreciation method?*
- SLN/DDB depreciation, asset register management, formula auditing
- **Deliverable**: Depreciation recommendation with live Q&A

## 🚀 Getting Started

### For Students & Teachers
**Visit the live platform:** https://bodangren.github.io/Business-Operations/

No installation required—the platform works directly in your browser with:
- No accounts or login required
- All progress saved locally to your device
- Full offline capability after initial load
- Mobile, tablet, and desktop support

### For Developers

```bash
# Clone the repository
git clone https://github.com/bodangren/Business-Operations.git
cd Business-Operations

# Navigate to the application
cd bus-math-nextjs

# Install dependencies
npm install

# Start development server
npm run dev

# Open in browser
# http://localhost:3000
```

### Build and Deploy
```bash
# Build static export
npm run build

# Run quality checks
npm run lint
npm run typecheck
npm test
```
