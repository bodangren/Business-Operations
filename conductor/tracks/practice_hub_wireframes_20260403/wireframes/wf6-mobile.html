Mobile Behavior Notes

Navigation Changes

Bottom tab bar replaces desktop header nav on mobile. Tabs: Home, Units, Practice, Glossary.
Practice tab is the primary entry to the practice hub on mobile (vs. header link on desktop).
Index moves to a "More" overflow menu or is accessed from the glossary page.

Practice Hub Home

Stats row becomes horizontally scrollable cards (not a grid) — swipe to see all stats.
Study mode cards become horizontally scrollable with larger tap targets (min 120px wide).
Two-column sections (Recent Practice + Weak Topics) stack vertically.
Unit filter becomes a dropdown or horizontal pill scroll instead of a button row.
Export section collapses to a single "Export Progress" button linking to the export page.

Flashcards

Full-width card fills the viewport width (no side margins). Min height 200px.
Buttons stack vertically instead of side-by-side — easier to tap with thumb.
Swipe gestures: swipe left = Review Again, swipe right = Got It. (Progressive enhancement — buttons always work.)
Keyboard hints hidden on mobile (no physical keyboard).

Matching Game

Terms and definitions stack vertically — not side-by-side columns.
Tap-to-select replaces drag-and-drop entirely on mobile. Select a term, then select its definition.
Matched pairs collapse out of view to reduce scrolling.

Speed Round

Answer grid becomes 1×4 vertical stack instead of 2×2 grid.
HUD (timer, score, streak) moves to a sticky bar at the top of the viewport.
Number keys hidden (no physical keyboard) — tap-only interaction.

Export Flow

Export type cards stack vertically (full width) instead of 3-column grid.
Preview table becomes horizontally scrollable or simplified to a list.
Step indicator collapses to a simple "Step 2 of 3" text label.

Progress Dashboard

Stats row becomes a 2×2 grid (or horizontal scroll).
Two-column sections (sessions + weak topics) stack vertically.
Unit mastery bars become compact rows with just unit abbreviation + bar + percentage.
Annotation — WF6 Key Decisions: Mobile breakpoint: 768px (standard Tailwind md breakpoint). Bottom tab navigation: Home, Units, Practice, Glossary. "More" overflow for Index, Export, Teacher pages. Horizontal scroll sections use scroll-snap for clean card-by-card browsing. Minimum tap target: 44px (Apple HIG guideline). Swipe gestures are progressive enhancement — always have button fallbacks. No separate mobile routes — responsive adaptation of the same routes.
