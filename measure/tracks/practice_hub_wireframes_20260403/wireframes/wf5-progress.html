Annotation — WF5 Key Decisions: Mastery bars use three color tiers: green (>75%), amber (50-75%), red (<50%). Unit rows link to unit-filtered practice hub view. "Export Progress" button links to /student/practice-hub/export. "Practice Weak Topics" starts a flashcard session pre-filtered to lowest-mastery terms. Session history shows last 10 sessions by default. All data from localStorage. Terms studied count vs total terms available (156 across all units).
Phase 3 — Actions, Data & Behavior: Actions: • Click unit row → /student/practice-hub?unit={id} (filtered hub view) • "Export Progress" button → /student/practice-hub/export • "Practice Weak Topics" → /student/practice-hub/flashcards?mode=weak (pre-filtered to lowest-mastery terms across all units) • Click session row → navigates to session's mode page with same unit filter (or shows read-only detail) Required Data: • termsStudied — count of unique termIds across all practice_terms_* keys • totalTermsAvailable — from glossary-data.ts total count (currently 156) • avgMastery — mean of all practice_terms_* masteryScore values • dueToday — count of terms where nextReviewDate ≤ today • totalSessions — length of practice_sessions[] • unitMastery[] — per-unit: { unitId, unitTitle, termsStudied, termsTotal, avgMastery } • sessions[] — last 10 from practice_sessions[], sorted by date desc • weakTopics[] — all terms with masteryScore < 60%, sorted ascending, limited to top 8 Local-Only: • All data computed from localStorage at page load • Mastery color tiers computed client-side: bar-high (>75%), bar-mid (50-75%), bar-low (<50%) Static Constraint: Dashboard is a static shell. All numbers hydrated client-side. Zero states for first-time visitors. Export Payload Relevance: This page is a consumer of the same data that WF4 exports. The "Export Progress" button triggers WF4 flow. The dashboard itself does not generate exports — it links to the export page. Open Decisions: • Should the dashboard show a chart/graph (e.g., mastery over time)? (Recommendation: not for MVP — bar charts are sufficient.) • Should "See all" link to a full session history page or paginate inline? (Recommendation: paginate inline with "Load more" for MVP.) • Average mastery: weighted by term count per unit or simple mean? (Recommendation: simple mean for MVP.)
